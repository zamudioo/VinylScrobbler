Let's set up VinylScrobbler
VinylScrobbler escucha tu tornamesa, identifica canciones con Shazam y las guarda en tu historial local.

Servicios opcionales: puedes vincular Last.fm para scrobbling y Spotify para agregar canciones a tus Liked Songs. Puedes saltar estos pasos y configurarlos después.

Lo único requerido es seleccionar tu dispositivo de audio.
Paso 1 de 4 — Opcional
Last.fm API Keys
Conecta Last.fm para scrobblear tus canciones automáticamente.
¿Sin Last.fm? No pasa nada — las canciones igual se guardan en la base de datos local y puedes ver todo en el dashboard. Puedes vincularlo después desde Settings.
Don't have API keys yet? Create a Last.fm API account →
Paso 2 de 4 — Opcional
Tu cuenta de Last.fm
Ingresa las credenciales de la cuenta que recibirá tus scrobbles.
Tu contraseña se hashea localmente (MD5) antes de enviarse. Nunca se guarda en texto plano.
Paso 3 de 4 — Opcional
Spotify
Conecta Spotify para agregar canciones detectadas a tus Liked Songs automáticamente.
Necesitas crear una app en Spotify Developer Dashboard y agregar este Redirect URI exacto:
http://127.0.0.1:8000/api/spotify/callback
🎧
Guarda las credenciales primero
Paso 4 de 4 — Requerido
Audio Input
Selecciona el dispositivo de audio conectado a tu tornamesa o mixer.
✓
All set!
VinylScrobbler is ready
Pon un disco y VinylScrobbler empieza a identificar canciones automáticamente.
